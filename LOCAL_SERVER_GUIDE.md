## 🔧 كيفية تشغيل النظام

### **الطريقة 1: تشغيل منفصل (مُوصى بها)**

```bash
# Terminal 1: تشغيل خادم المعاينة
npm run preview -- --host

# Terminal 2: تشغيل خادم SMTP
node simple-smtp-server.js
```

### **الطريقة 2: تشغيل متزامن**

```bash
# تشغيل كلا الخادمين معاً
npm run dev
```
