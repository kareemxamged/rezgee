### 2. تحديث مكون صفحة تسجيل الدخول (`src/components/LoginPage.tsx`)

#### التغييرات الرئيسية:
1. **إضافة دعم الترجمة:**
   - استيراد `useTranslation` من `react-i18next`
   - استخدام `const { t, i18n } = useTranslation()`
   - تحويل جميع النصوص الثابتة لاستخدام `t()`

2. **ضبط الاتجاهات (RTL/LTR):**
   - تحديث `dir` attribute ليكون ديناميكي: `dir={i18n.language === 'ar' ? 'rtl' : 'ltr'}`
   - ضبط مواضع الأيقونات حسب اللغة
   - ضبط padding للحقول حسب الاتجاه
   - ضبط اتجاه الأسهم في الأزرار

3. **تحديث schema التحقق:**
   - تحويل رسائل التحقق لاستخدام الترجمة
   - جعل schema ديناميكي داخل المكون

#### أمثلة على التحديثات:

**ضبط مواضع الأيقونات:**
```tsx
<Mail className={`absolute ${i18n.language === 'ar' ? 'right-3' : 'left-3'} top-1/2 transform -translate-y-1/2 text-slate-400 w-5 h-5`} />
```

**ضبط padding للحقول:**
```tsx
className={`w-full px-4 py-3 ${i18n.language === 'ar' ? 'pr-12' : 'pl-12'} border border-slate-300 rounded-xl focus:ring-2 focus:ring-primary-500 focus:border-transparent transition-all duration-200 bg-white/50`}
```

**ضبط اتجاه الأسهم:**
```tsx
<ArrowRight className={`w-5 h-5 ${i18n.language === 'ar' ? '' : 'rotate-180'}`} />
```

### 3. الميزات المحققة

#### ✅ الترجمة الكاملة
- جميع النصوص مترجمة ومتاحة بالعربية والإنجليزية
- رسائل التحقق والأخطاء مترجمة
- النصوص التفاعلية (أزرار، روابط) مترجمة

#### ✅ دعم RTL/LTR

- مواضع الأيقونات تتكيف مع الاتجاه
- padding وmargin محسوبة بشكل صحيح
- اتجاه الأسهم يتكيف مع اللغة

#### ✅ تجربة مستخدم محسنة
- تبديل سلس بين اللغات
- تخطيط متسق في كلا الاتجاهين
- حفظ تفضيلات المستخدم
