🔍 تشخيص مشكلة علامات الصح

تحليل مفصل لمشكلة عدم ظهور علامات الصح الزرقاء داخل المحادثة

🐛 خطوات التشخيص

  1. الخطوة 1: افتح أدوات المطور في المتصفح (F12)
  2. الخطوة 2: اذهب إلى تبويب Console
  3. الخطوة 3: قم بإرسال رسالة من حساب لآخر
  4. الخطوة 4: اقرأ الرسالة من الحساب المستقبل
  5. الخطوة 5: ارجع للحساب المرسل وافتح المحادثة
  6. الخطوة 6: ابحث في الكونسول عن الرسائل التالية:

📊 رسائل الكونسول المتوقعة:

1. عند تحميل الرسائل:

Loaded messages: [...] Messages read_at status: [ { id: "message-id", sender_id: "sender-id", read_at: "2025-08-09T...", // يجب أن يحتوي على تاريخ isMyMessage: true } ]

2. عند عرض كل رسالة:

getDeliveryStatusIcon for message: { id: "message-id", delivery_status: undefined, read_at: "2025-08-09T...", // يجب أن يحتوي على تاريخ hasReadAt: true // يجب أن يكون true }

🚨 إذا كانت المشكلة مستمرة:

إذا كان read_at يظهر null في الكونسول، فهذا يعني أن:

  • دالة markMessagesAsRead لا تعمل بشكل صحيح
  • أو أن هناك مشكلة في استعلام قاعدة البيانات
  • أو أن الرسائل لم يتم وضع علامة "مقروء" عليها فعلياً

🔧 الحلول المحتملة

الحل 1: التحقق من دالة markMessagesAsRead

// في الكونسول، تحقق من هذه الرسالة: "Messages marked as read for conversation: conversation-id" // إذا لم تظهر، فهناك مشكلة في دالة markMessagesAsRead

الحل 2: فحص قاعدة البيانات مباشرة

-- في Supabase SQL Editor SELECT id, sender_id, content, read_at, created_at FROM messages WHERE conversation_id = 'bb8863a2-20c9-4da4-b1cb-7530cb876217' ORDER BY created_at DESC;

الحل 3: إعادة جلب الرسائل بعد التحديث

// إضافة إعادة جلب الرسائل بعد markMessagesAsRead if (markResult.success) { // إعادة جلب الرسائل من قاعدة البيانات للحصول على read_at المحدث const { data: updatedMessages } = await messageService.getMessages(conversationId); if (updatedMessages) { setMessages(updatedMessages.map(msg => ({ ...msg, sender: Array.isArray(msg.sender) ? msg.sender[0] : msg.sender }))); } }

🎯 النتيجة المتوقعة

بعد تطبيق التشخيص والإصلاحات:

  • ✅ read_at يحتوي على تاريخ صحيح في الكونسول
  • ✅ hasReadAt يظهر true للرسائل المقروءة
  • ✅ علامات الصح الزرقاء تظهر داخل المحادثة
  • ✅ علامات الصح الزرقاء تظهر في قائمة المحادثات
